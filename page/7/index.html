用php计算wav文件播放时间

2013年05月28日

开发需要，试写了一个php版的

代码如下：

/**
 * get length of wav audio in minutes/seconds
 *
 * @package default
 * @author linyang95#aol.com
 **/
$filename = 'voice_0.wav';
$fp = fopen($filename, 'r');
fseek($fp, 20);
$rawheader = fread($fp, 16);
// v short string
// V long string
$header = unpack('vwFormatTag/vnChannels/VnSamplesPerSec/VnAvgBytesPerSec/vnBlockAlign/vwBitsPerSample', $rawheader);
extract($header);
$rawbody = fread($fp, filesize($filename)-36);
$second = strlen($rawbody) / ( ($nChannels * $nSamplesPerSec * $wBitsPerSample) / 8 );
print ceil($second);

mysql_config not found on osx10.8

2013年05月23日

系统：osx 10.8.3

pip install mysql-python

抛出异常

EnvironmentError: mysql_config not found

由于mysql是macport安装，版本55，所以问题就简单了

只需运行命令

sudo ln -s /opt/local/bin/mysql_config5 /opt/local/bin/mysql_config
