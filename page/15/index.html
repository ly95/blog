Git svn rebase异常

2014年08月22日

环境MacOS，git版本1.9.3(Apple Git-50)，同步版本信息时遭遇此错误提示

$ git svn rebase

Incomplete data: Delta source ended unexpectedly at /Applications/Xcode.app/Contents/Developer/usr/share/git-core/perl/Git/SVN/Ra.pm line 290.

处理办法指定同步版本

git svn reset —r 版本号

然后再试

git svn rebase

如果还报错，重试指定较低的版本。
